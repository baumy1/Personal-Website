WB Messenger

WB Messenger was a basic 'instant messenger' I made as a learning experience that ended up being used by my peers to talk at school. It included login authentication, XSS and MySQL injection prevention and you could send images and messages. WB Messenger was written in PHP, HTML, CSS and used a MySQL database.
